## Cool plugins and helper soft

###  Animating for music
[Animation nodes](https://animation-nodes.com/) - Blender 4.2 or 4.3 (or compile for the last Python version somehow)

[MIDIController](https://extensions.blender.org/add-ons/midicontroller/) - Real time MIDI input 

### Utility soft


[MidiEditor (WIN only)](http://www.midieditor.org/) - A standalone tool to inspect MIDI files when MIDI-to-animation mapping. 

[PolyCam](https://poly.cam/) - used to be a solid mobile 3d scanning app. Now only has so many free mesh export options. 

[InstantMeshes](https://github.com/wjakob/instant-meshes) - Retopology tool. Good for 3D scan postprocessing


## Learning resources

Below is the list of legends, that are famous for their unique ways of getting things done, alongside just speaking well:
- Ian Hubert's "Lazy Tutorial" series - tons of quick'n'dirty tricks for <b> modelling,compositing and animation</b> that work damn well  

    [Ian Hubert YT](https://www.youtube.com/@IanHubert2/videos)

- CGMatter  - <b>shading and compositing etc</b>. - mostly non-realtime stuff. Advanced ideas to learn from. 

    [CGMatter YT](https://www.youtube.com/@CGMatter)

    [Default Cube (secong channel; longer tutorials)](https://www.youtube.com/@DefaultCube/videos)

- Blender Guru - the author of the famous 'Donut tutorial'. The following two courses build a solid basis of <b>modelling techniques</b> for any target - whether photorealism or game. BG's channel is also a good go-to for photorealism techniques.

    [Donut tutorial](https://www.youtube.com/watch?v=-tbSCMbJA6o&list=PLjEaoINr3zgGUwGwXlj9kBe7TrVWNjkyv) - learn basic modelling toolset, incl. basic shading and particle systems
    
    [Anvil tutorial](https://www.youtube.com/watch?v=yi87Dap_WOc&list=PLjEaoINr3zgHJVJF3T3CFUAZ6z11jKg6a) - good intermediate tutorial, incl. subdivision workflow, sculpting, texture baking, high-to-low poly workflow.
